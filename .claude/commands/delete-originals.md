---
description: originals/ 配下のフォルダの中身を空にする（作業用フルサイズ画像の後片付け）
argument-hint: "[slug]（省略時は originals/ 内すべてのフォルダ）"
allowed-tools: Bash(ls:*), Bash(find:*), Bash(rm:*), Bash(du:*), Bash(test:*)
---

`originals/` 内のフォルダの中身を空にする後片付けタスクです。R2 へのアップロードや
サムネ・モックアップ生成が済んだあと、ローカルの作業用フルサイズ画像を削除して容量を戻します。

- 対象: `$1` が指定されていれば `originals/$1` のみ。未指定なら `originals/` 直下の全フォルダ。

次の手順で進めてください。

1. **対象の確認**
   - `originals/` が無い、または空なら「削除するものはありません」と伝えて終了。
   - `$1` 指定時は `originals/$1` が存在するか確認。無ければ知らせて終了。
   - `find <対象> -type f` で、消す予定のファイル一覧と件数、`du -sh` で概算サイズを表示する。

2. **削除実行（中身のみ・フォルダは残す）**
   - 各対象フォルダについて、その**中身だけ**を削除しフォルダ自体は空で残す。例:
     ```
     for d in originals/*/; do rm -rf "$d"*; done      # 全フォルダ
     rm -rf originals/$1/*                             # slug 指定時
     ```
   - **絶対に `originals/` の外には触れないこと。** `thumbs/`・`mockups/`・`data/` 等は削除しない。

3. **結果報告**
   - 各フォルダが空になったことを `find originals -type f | wc -l`（0 になる）で確認し、
     空にしたフォルダ名と、解放したおおよその容量を報告する。

注意：
- これはローカルの作業ファイルの削除で、元に戻せない。ただし `originals/` は `.gitignore` 済みで
  Git 管理外なので、リポジトリやサイトには影響しない。
- R2 未アップロードのフォルダを消すと再取得が必要になるため、手順1で一覧を見せ、明らかに未処理
  （例: R2 に上がっていない）と分かる場合は実行前に一言確認する。
